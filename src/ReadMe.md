### Name of your project
 KeenKeeper - Smart Friend Management System

### A short description
KeenKeeper is a modern web application designed to help you maintain relationships with your friends. It tracks your last contact dates and reminds you who is overdue for a check-in through an intuitive dashboard.

---

###  Technologies Used
I have used the following technologies to build this project:

- *Frontend:* Next.js 
- *Styling:* Tailwind CSS & DaisyUI
- *Icons:* React Icon
- *Charts:* Recharts
- *Notifications:* React tostfy
- *Deployment:* Vercel
